Codewars Challenges & Solutions

This project is a collection of Codewars challenges I’ve worked through to improve my JavaScript skills and problem-solving ability. Rather than just storing final answers, this repo shows my thinking process and how my solutions evolve over time.

How It’s Made

Tech used: HTML, CSS, JavaScript

Most challenges start with a solution that makes the logic clear, even if it’s not the most concise way to solve the problem. After that, I usually include an alternative approach—either a simpler version or a more compact solution—to show how the same problem can be solved more efficiently.

This helped me focus on understanding why a solution works before worrying about making it clever. Comparing different approaches side-by-side made it easier to spot patterns, learn new JavaScript techniques, and improve code readability.

⸻

Optimizations

For many problems, I revisited my original solution and refactored it into a cleaner or more efficient version. This often meant reducing extra steps, using built-in JavaScript methods, or rewriting the logic in a more direct way.

Highlighting both versions shows how my thinking has improved and demonstrates that I care about writing better code—not just code that works.

⸻

Lessons Learned

One of the biggest lessons from this project is that the first solution isn’t always the best one—and that’s okay. Writing a working solution first, then refining it, made learning much more effective.

This approach helped me become more comfortable refactoring, thinking critically about my own code, and continuously improving. It also reinforced that growth as a developer comes from iteration, not perfection.
