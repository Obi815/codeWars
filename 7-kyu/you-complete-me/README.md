<h2><a href=https://www.codewars.com/kata/63967eb85e54bd00237d2201/train/javascript target="_blank">You Complete Me</a></h2><h3>7 kyu</h3><p>Task: concatenate one or more letters to the end of a given input string tocreate a string that reads the same  forward as backward. </p><p> Specifications</p><ul><li><p>Letters can only be added to the right hand side of the input string. </p></li><li><p>You are required to add at least one letter to the input string. </p></li><li><p>The shorter return string is the correct one. ie.. (ab)=&gt; abba is Wrong, while  (ab)=&gt; aba is Right. </p></li><li><p>Inputs will consist of one or more letters, upper and lower case. </p></li><li><p>Strings are case sensitive. ie..(Gn)=&gt; gng is Wrong, while (Gn)=&gt; GnG is right.</p></li></ul><p> Examples: </p><p>(a) =&gt; a + a =&gt; aa </p><p>(GG) =&gt; GG + G =&gt; GGG </p><p>(Ab) =&gt; Ab + A =&gt; AbA </p><p>(aba) =&gt; aba + ba =&gt; ababa </p><p>(aab) =&gt; aab + aa =&gt; aabaa </p>
